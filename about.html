The Minecraft Spark

My journey into programming began unexpectedly while tinkering with Minecraft servers. What started as a simple desire to create my own gaming world turned into something much more fascinating. I found myself diving into Notepad, meticulously customizing server settings line by line, building everything from scratch. The thrill of seeing my modifications come to life in the game ignited a passion for coding that I never knew existed. This simple start opened my eyes to the vast possibilities of programming, where every line of code could create something tangible and interactive.
