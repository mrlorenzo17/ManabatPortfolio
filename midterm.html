One of the topics that stood out for me in my midterm journey is the spacy library, it was so powerful and efficient when it comes to building applications that process and "understand" large volumes of text. spaCy is a free, open-source library for advanced NLP in python which made it easier for me to actually apply it to some works that I have made throughout the journey. The way install spaCy is by typing pip install spacy in python.

I've also learned how to evaluate nlp models that was discussed in lecture 6. Being able to clearly answer questions that is relevant to the goal of your model. There are different types of evaluation and metrics for each goals. One of them is the simplest performance metric: accuracy, where accuracy is just a percentage of correctly predicted instances. But accuracy only works well when the number of task instances is fixed, when it has exactly one correct answer and all errors are equally significant. If the conditions are not met there comes the Precision and recall. These two are used in evaluating classification models where the data is imbalanced, or where the consequences of different types of errors vary. Precisions is only if our systems gives a quality prediction and recall is the average of all possible correct outputted by the system. Those were applied in some works that I've done through the journey.
